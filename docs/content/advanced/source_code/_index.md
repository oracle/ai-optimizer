The {{% short_app_ref %}} can export your tested configuration as a downloadable source code template, ready to run as a standalone application. The Spring AI template exposes the RAG chatbot as an OpenAI API-compatible REST endpoint.

Templates are available from the [Settings]({{% relref "/client/configuration/settings" %}}) page when your configuration uses either Ollama or OpenAI as providers for both chat and embedding models.
